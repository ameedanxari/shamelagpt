# ADR-008: Emerald + Amber Brand Color Scheme

**Status**: Accepted
**Date**: 2025-11-02

## Context
Need consistent visual identity across platforms with light/dark mode support.

## Decision
Use **Emerald (#10B981) + Amber (#F59E0B)** as primary brand colors:
- Emerald: Primary actions, success states
- Amber: Accents, highlights, warnings
- Both light and dark mode palettes defined

## Consequences
**Positive:**
- Consistent branding across platforms
- Accessible color contrast
- Works well with both light/dark modes
- Memorable visual identity
